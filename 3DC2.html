Supervising Swarm Intelligence

We are working with Eyes3Shut and Télécom Bretagne to build interactive touch table systems that allow several experts to works with all their data on the same work surface. Custom smart glasses allow to separate private data from shared objects viewed by all.

Immediate interaction around a table allows efficient decision making and avoids misunderstandings.
